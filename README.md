# Weather-App
This Weather App allows you to enter the name of any city in the search bar and instantly get the current temperature and detailed weather information for that location. The app fetches real-time data from the OpenWeatherMap API to provide accurate and up-to-date weather details. 

In addition to the current temperature, it also displays an hourly forecast, showing anticipated temperature changes and weather conditions throughout the day. Designed with a simple and user-friendly interface, this app makes it easy to check both current and hourly weather anywhere in the world. This project is ideal for learning API integration and front-end development with practical applications.
